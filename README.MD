### Інструкція з встановлення
Після клонування репозиторію необхідно переіменувати файл `appsettings_sample.json`
до `appsettings.json`, 
- відкрити його властивості та встановити пункт **Copy to output directory** у значення **Copy always**
- відредагувати зміст файлу, замінити плесхолдери *** на актуальні дані

Додати до проєкту базу даних (або підключити таку, що існує)
- якщо БД немає, то ПКМ (проєкт) - Add - New Item - [Data] Service Based Database
- корегуємо рядок підключення до БД у файлі `appsettings.json` (DblClk .mdf, Server Explorer -> Properties)
